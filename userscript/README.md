# DeAdFly userscript

DeAdFly userscript does exactly the same as the extension, minus AdFly Redir.

## Compatibility

* Firefox 12.0 or newer
* SeaMonkey 2.9 or newer

* Greasemonkey 1.0 or newer
* Scriptish 0.1.8 or newer
