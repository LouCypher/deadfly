# DeAdFly

Expand AdF.ly link via Firefox context menu.

## Compatibility

* Firefox 10.0 - 21.0a2
* SeaMonkey 2.7 - 2.18a2

## Installations

* [Latest dev build](http://dl.dropbox.com/u/120550/extensions/deadfly/deadfly.xpi).
